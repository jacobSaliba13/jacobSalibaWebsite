LIQUID

Essentially the “show” can be recreated in multiple ways mainly having a prerecorded format, A live format which is projected through the use of an overhead projector as seen in the project Music/Light and a live format which uses software as to manipulate the images composed in real-time. Each format has its own advantages and disadvantaged which are analysed for each scenario. The beauty of each liquid light show is that of using and trying to control organic forms as to create this controlled yet spontaneous work which complements the music.
