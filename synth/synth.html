Synth

Synth is my disertation project and is an ongoing project which explores the history of audio visual performance, perception theories and emersion in combination to space and synthesis. The final aim of the question is that of creating a software based synth which enables the muscian to create a visual performance in combination of his musical live performance. This will be achieved by using Touchdesigner a node based programming language and Abelton.
